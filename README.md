
#Proof-of-Popularity
Requirement:
  MetaMask. 

Steps:
1). clone the repository run:
`npm start`

2). Set your MetaMask to use the Kovan Test Network.

See `popularity.sol` for the smart contract (hosted on Kovan Testnet).
